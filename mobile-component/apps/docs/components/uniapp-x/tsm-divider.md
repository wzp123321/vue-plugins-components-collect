## Props

| text | 分割线文字 | string | 否 | - |
| color | 分割线颜色 | string | 否 | '#dcdfe6' |
| contentPosition | 文字位置 | 'left' \| 'center' \| 'right' | 否 | 'center' |
| dashed | 是否虚线 | boolean | 否 | false |
| customClass | 自定义类名 | string | 否 | - |
| customStyle | 自定义样式 | object | 否 | {} |
