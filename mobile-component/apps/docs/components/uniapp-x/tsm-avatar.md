## Props

| 属性名 | 说明 | 类型 | 是否必填 | 默认值 |
| --- | --- | --- | --- | --- |
| src | 头像图片地址 | string | 否 | - |
| shape | 头像形状 | 'circle' \| 'square' | 否 | 'circle' |
| size | 头像大小 | number \| string | 否 | 40 |
| text | 头像文字 | string | 否 | - |
| bgColor | 背景颜色 | string | 否 | '#c0c4cc' |
| color | 文字颜色 | string | 否 | '#ffffff' |
| customClass | 自定义类名 | string | 否 | - |
| customStyle | 自定义样式 | object | 否 | {} |
