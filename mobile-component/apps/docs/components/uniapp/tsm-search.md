## Props

| 属性名 | 说明 | 类型 | 是否必填 | 默认值 |
| --- | --- | --- | --- | --- |
| value | 搜索框值 | string | 否 | - |
| placeholder | 占位符 | string | 否 | '请输入搜索关键词' |
| clearable | 是否显示清除按钮 | boolean | 否 | true |
| showAction | 是否显示搜索按钮 | boolean | 否 | true |
| actionText | 搜索按钮文字 | string | 否 | '搜索' |
| customClass | 自定义类名 | string | 否 | - |
| customStyle | 自定义样式 | object | 否 | {} |
