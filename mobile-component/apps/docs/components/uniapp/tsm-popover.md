## Props

| 属性名 | 说明 | 类型 | 是否必填 | 默认值 |
| --- | --- | --- | --- | --- |
| show | 是否显示 | boolean | 否 | false |
| placement | 弹出位置 | 'top' \| 'bottom' \| 'left' \| 'right' | 否 | 'bottom' |
| title | 标题 | string | 否 | - |
| customClass | 自定义类名 | string | 否 | - |
| customStyle | 自定义样式 | object | 否 | {} |
