## Props

| 属性名 | 说明 | 类型 | 是否必填 | 默认值 |
| --- | --- | --- | --- | --- |
| value | 进度值 | number | 否 | 0 |
| size | 进度条大小 | number \| string | 否 | 100 |
| strokeWidth | 进度条宽度 | number | 否 | 6 |
| color | 进度条颜色 | string | 否 | '#2979ff' |
| bgColor | 背景颜色 | string | 否 | '#ebedf0' |
| showText | 是否显示文字 | boolean | 否 | true |
| customClass | 自定义类名 | string | 否 | - |
| customStyle | 自定义样式 | object | 否 | {} |
