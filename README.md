# npm&pnpm 命令对比

| npm 命令          | pnpm 命令       | 用途         |
| ----------------- | --------------- | ------------ |
| npm install       | pnpm install    | 安装全部依赖 |
| npm install xxx   | pnpm add xxx    | 安装指定依赖 |
| npm uninstall xxx | pnpm remove xxx | 移除指定依赖 |
| npm run xxx       | pnpm xxx        | 运行脚本     |
