# 样式规范

## SCSS使用规范

- 所有样式文件使用 `.scss` 扩展名
- 支持全局样式和模块化样式
- 使用嵌套语法，但避免过深的嵌套层级（建议不超过3层）
- 使用变量和混合器(mixins)提高代码复用性

## 文件组织

```
src/styles/
├── variables.scss      # 全局变量
├── mixins.scss        # 混合器
├── common.scss        # 公共样式
├── element-plus.scss  # Element Plus主题定制
└── index.scss         # 主入口文件
```

## 主题变量

### Element Plus变量覆盖

```scss
// src/styles/element-plus.scss
// 覆盖Element Plus默认变量
:root {
  --el-color-primary: #409eff;
  --el-color-success: #67c23a;
  --el-color-warning: #e6a23c;
  --el-color-danger: #f56c6c;
  --el-color-info: #909399;

  // 自定义主题色
  --user-primary-color: #1890ff;
  --user-secondary-color: #52c41a;
  --user-border-radius: 4px;
}
```

### 自定义变量

```scss
// src/styles/variables.scss
// 布局变量
$layout-header-height: 60px;
$layout-sidebar-width: 240px;
$layout-content-padding: 24px;

// 颜色变量
$primary-color: var(--el-color-primary);
$success-color: var(--el-color-success);
$warning-color: var(--el-color-warning);
$error-color: var(--el-color-danger);
$info-color: var(--el-color-info);

// 字体变量
$font-size-base: 14px;
$font-size-lg: 16px;
$font-size-sm: 12px;

// 间距变量
$spacing-xs: 4px;
$spacing-sm: 8px;
$spacing-md: 16px;
$spacing-lg: 24px;
$spacing-xl: 32px;
```

## 响应式设计

- 使用移动端优先的设计思路
- 断点定义：
  - 超小屏幕（手机）：< 768px
  - 小屏幕（平板）：≥ 768px
  - 中等屏幕（桌面）：≥ 992px
  - 大屏幕（大桌面）：≥ 1200px

```scss
// 媒体查询示例
.user-card {
  // 移动端样式
  padding: $spacing-sm;

  @media (min-width: 768px) {
    // 平板及以上样式
    padding: $spacing-md;
  }

  @media (min-width: 1200px) {
    // 大桌面样式
    padding: $spacing-lg;
  }
}
```

## 样式命名规范

- 使用BEM命名法（Block\_\_Element--Modifier）
- 组件根元素使用组件名作为block名
- 子元素使用双下划线连接
- 修饰符使用双连字符连接

```scss
// BEM命名示例
.user-card {
  // Block
  &__header {
    // Element
    &--active {
      // Modifier
      background-color: $primary-color;
    }
  }

  &__title {
    // Element
    font-size: $font-size-lg;
  }
}
```

## 性能优化

- 避免使用深层嵌套选择器
- 合理使用scoped样式
- 提取公共样式到mixins
- 使用CSS变量提高主题切换性能

## 主题切换

支持亮色和暗色主题：

```scss
// 主题变量定义
:root {
  // 亮色主题
  --bg-color: #ffffff;
  --text-color: #333333;
  --border-color: #dcdfe6;
}

[data-theme='dark'] {
  // 暗色主题
  --bg-color: #141414;
  --text-color: #ffffff;
  --border-color: #434343;
}
```
