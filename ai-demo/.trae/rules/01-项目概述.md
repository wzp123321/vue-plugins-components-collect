# 项目概述

## 项目定位

一个基于 Vue3 + TypeScript 的单页应用（SPA）。

## 技术栈

| 领域      | 技术                       | 说明                                                      |
| --------- | -------------------------- | --------------------------------------------------------- |
| UI 框架   | Vue 3.5.x                  | 强制使用 Composition API                                  |
| 类型系统  | TypeScript 5.x             | 强制使用，禁止 JavaScript                                 |
| 构建工具  | Vite 7.x                   | 直接配置使用                                              |
| 路由管理  | Vue Router 4.x             | 强制使用                                                  |
| 状态管理  | Pinia 2.x                  | 统一使用，禁止其他方案                                    |
| 组件库    | @tiansu/element-plus 2.4.x | 所有交互 UI 必须基于 Element Plus 二次封装，使用 te- 前缀 |
| 样式方案  | SCSS                       | 强制使用，支持全局样式和模块化样式                        |
| HTTP 请求 | Axios 1.6.x                | 需要基于 axios 进行封装，对请求和响应进行拦截处理         |
| 工具函数  | lodash-es 4.17.x           | 编写或封装函数时，优先考虑是否有可使用的 lodash 函数      |
| 时间工具  | dayjs 1.11.x               | 涉及日期时间的处理统一用 dayjs                            |
| 开发工具  | @vueuse/core 10.x          | Vue Composition API 工具库                                |
